As an admin user I want to be able to login into the pathfinder application
As an admin user I want to be able to create and update customers
As an admin user I want to be able to create and update pathfinder users 
As an admin user I want to be able to assign users to customers
As an admin user I want to be able to assign users to roles e.g. business owner, application assessor, application owner
As a user I want to be able to login into the pathfinder application
As a business owner I want to be able to add applications which will be assessed for a customer
As a user I want to be able to list all applications for a customer
As a business owner or application owner I want to be able to add customer applications/systems which will not be assessed but used as assessment dependencies
As a business owner I want to be able to see the overall progress of the customers assessment work with the count of the number of applications assessed, in progress and not started
As a business owner I want to be able to assign an application assessment to an application owner
As an application assessor I want to be able to select from a list the application that I will assess
As an application assessor I want to be able to perform the application accessment with the application owner and answer the multiple-choice assessment questions
As an application assessor I want to be able to capture free-form information not covered by the existing applications assessment questions
As an application assessor I want to be able to change answers in assessments already started but not completed
As an application assessor I want to be able to view extra help on assesssment questions in case of confusion over the understanding of the questions
As an application assessor I want to be able to clone a previous completed application assessment responses in order to handle similar applications
As an application assessor in conjunction with the application owner I want to be able to designate an application to be cloud native, suitable for migration (lift and shift), suitable for modernisation (some rewrite) or not-container suitable 
As an application assessor in conjunction with the application owner I want to be able to assign a t-shirt sized work effort rating for each application be it migration or modernisation 
As an application assessor in conjunction with the application owner I want to be able to add assessment notes, decisions and recommendations to the application assessment
As a business owner I want to be able to assign each application a business criticality rating 
As a business owner I want to be able to view the breakdown of applications into cloud native, modernisable and unsuitable for containers
As a business owner I want to be able to view the breakdown of applications by work effort
As a business owner I want to be able to view the breakdown of applications by business priority
As a business owner I want to be able to view the breakdown of applications by dependency order
As a business owner I want to be able to view the entire application landscape in categories of cloud native, modernisable and unsuitable for containers 
As a business owner or application owner I want to be able to view a summary of each application with the main findings and assessment decisions
As a business or application owner I want to be able to view application asssessments which have questions that have "unknown" answers
As a business owner I want to be able to export the data container in the application assessment
